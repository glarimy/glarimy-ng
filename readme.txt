WORKING WITH ANGULAR 2+
0. Install Node and NPM
1. Install Angular CLI
npm install -g @angular/cli
2. Create new project
ng new <project-name>
3. Move to <project-name> and run the project
ng serve
4. Verify output on the browser
http://localhost:4200

IMPORTANT NG COMMANDS
ng new <project> 
ng serve
ng test
ng ete
ng generate module <module-name>
ng generate module <module-name> --routing
ng generate component <component-name> --module <module-name>
ng generate service <service-name> --module <module-name>
